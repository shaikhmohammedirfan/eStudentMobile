Register New Student

1. From Home page tap on Register button

2. Enter School key & Student key and tap on Submit button

3. Student details will appear on the screen

4. Review and confirm registration by tapping on Register button

5. Registration is complete

** Note: Contact School in case the student details are incorrect or require any modification

News And Events

1. From Main Menu select "News" or from Student Dashboard tap on "News" button

2. List of News & Events will be displayed

3. Tap on desired News/Event item to view details

** Note: Contact School in case the student details are incorrect or require any modification

Student Dashboard

1. From home page, tap on student name

2. Student Dashboard will be displayed on screen

Student Profile

1. From home page, tap on student name

2. Student Dashboard will be displayed on screen

3. Tap on Student profile to view student profile/details

Student Reports

1. From home page, tap on student name

2. Student Dashboard will be displayed on screen

3. Tap on Reports to view list of student reports

4. Tap on the required button to view desired report card

School Diary

1. From home page, tap on student name

2. Student Dashboard will be displayed on screen

3. Tap on Diary to view list of various fixtures, schedules and events

** Use Today, Weekly or Monthly view by tapping on desired button
